# 114-2 平行與分散式程式設計 — 期末模板

期末專題所需的 LaTeX 模板:論文報告與 peer review 評分表。

## 檔案

- `paper_template.tex` — IEEE 會議論文格式的期末報告模板。
- `review_criteria.tex` — 用於審查的一頁式評分表。

## 使用方式

兩個檔案皆以 **pdfLaTeX** 編譯。建議使用方式:開啟 [Overleaf](https://www.overleaf.com),新建專案,上傳 `.tex` 檔後編譯即可。Overleaf 已內建 `IEEEtran` class,不需額外檔案。

## 論文模板說明

- 在標題頁**勾選專題類別(A 或 B)**,讓審查者套用正確的評分視角:
  - **A** — 自訂題目。
  - **B** — 以 AI Agent 進行程式碼優化。
- **頁數限制:6 頁(含參考文獻)。** Appendix 的原始資料**不計入**頁數限制。
- 撰寫時請把每一處灰色的 `[方括號]` 提示換成自己的內容,並在繳交前刪除(包含頁數限制的提示文字)。
- **B 類**必須在 Appendix 附上原始資料:input prompt、system prompt、代表性的 agent output,以及對 agent 所做的更改。若內容過長,可改為提供 repository 連結。(A 類:Appendix 為選填。)
- 每次編譯會在 log 印出一行 `[PAGE BUDGET]`,顯示正文 + 參考文獻結束在第幾頁,方便檢查 6 頁限制。

## 重要時程

- **論文繳交:** 6/15(一)中午前。
- **同儕審查:** 6/15 中午 ~ 6/18 中午。每位同學審查 2 篇其他類別、2 篇同類別論文(由助教亂數分配)。
